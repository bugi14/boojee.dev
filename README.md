# boojee.dev

This is Darren Buttigieg's personal website.

## Local development

Install dependencies, then start the Vite dev server:

```sh
npm install
npm run dev
```

This serves the site locally (Vite will print the local URL, typically `http://localhost:5173`) with hot module reloading.

Other available scripts:

- `npm run build` — build the production bundle to `dist/`.
- `npm run preview` — serve the production build locally to sanity-check it before deploying.
